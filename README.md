# Informe Semanal (InfyCredit & Visionamos)

## Cómo usar
1) Coloca tus logos en `assets/`:
   - `assets/infycredit.png`
   - `assets/visionamos.png`

2) Abre `index.html` en el navegador, o súbelo a GitHub Pages.

## Archivos
- `index.html` (estructura)
- `styles.css` (responsive/UX)
- `app.js` (filtros + ordenamiento)
- `data.js` (datos exportados del Excel)
